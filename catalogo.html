Catálogo de Perguntas

Como ligar um LED com Arduino?
Use resistor de 220Ω, conecte o anodo ao pino digital e o catodo ao GND.
Qual a diferença entre servo e motor DC?
Servo controla ângulo com precisão. Motor DC gira continuamente.
Como usar sensor ultrassônico HC-SR04?
Trig → pulso de 10µs, Echo → mede tempo de retorno do pulso.
Como usar um botão com Arduino?
Conecte uma perna ao 5V, outra ao pino digital + resistor pull-down 10kΩ ao GND.
O que é PWM e como usar?
PWM simula sinal analógico. Use analogWrite(pino, 0-255) em pinos com ~.
Como ler sensor de temperatura LM35?
Conecte saída ao pino analógico. Valor = analogRead(A0) * 0.488 → °C.
Como controlar um relé com Arduino?
Use transistor NPN (ex: BC547) ou módulo relé. Acione com digitalWrite().
Qual a diferença entre analogRead e digitalRead?
analogRead: 0-1023 (10 bits). digitalRead: HIGH ou LOW.
Como usar display LCD 16x2 com I2C?
Use biblioteca LiquidCrystal_I2C. Endereço comum: 0x27 ou 0x3F.
Como conectar múltiplos sensores?
Use pinos diferentes ou barramento I2C (mesmo endereço? Mude com jumper A0/A1).
Como usar o Serial Monitor?
No setup(): Serial.begin(9600). Envie com Serial.println().
Como evitar "ruído" em sensores analógicos?
Use capacitor 0.1µF entre pino e GND + média de várias leituras.
Como usar buzzer com Arduino?
Conecte perna longa ao pino digital, curta ao GND. Use tone(pino, frequência).
Como fazer um LED piscar?
Use digitalWrite(LED, HIGH); delay(500); digitalWrite(LED, LOW); delay(500);.
Como usar potenciômetro?
Conecte terminal central ao pino analógico (A0). Outros dois: 5V e GND.
Como usar sensor de luz (LDR)?
Divisor de tensão com resistor 10kΩ. Leia com analogRead().
Como usar módulo Bluetooth HC-05?
Conecte TX → RX Arduino, RX → TX. Use SoftwareSerial.
Como usar cartão SD com Arduino?
Use biblioteca SD. Pino CS geralmente no 4 (Uno) ou 10 (Mega).
Como usar motor de passo?
Use driver ULN2003 ou A4988. Controle com biblioteca Stepper.
Como usar sensor PIR (movimento)?
Conecte saída ao pino digital. Leia com digitalRead(). Tem delay interno.
Como usar joystick analógico?
X → A0, Y → A1, SW (botão) → pino digital com pull-up.
Como usar sensor de umidade do solo?
Conecte saída analógica ao A0. Valores: seco (~1023), úmido (~300).
Como usar RTC DS3231?
Conecte via I2C. Use biblioteca RTClib. Muito preciso!
Como usar matriz de LED 8x8?
Use driver MAX7219. Biblioteca LedControl. Fácil de animar!
Como usar ESP8266 com Arduino?
Conecte via Serial (TX/RX). Use AT commands ou como WiFi shield.
Como usar sensor de gás MQ-2?
Conecte AO ao pino analógico. Calibre em ar limpo antes.
Como usar interrupt com botão?
Use attachInterrupt(digitalPinToInterrupt(pino), funcao, RISING).
Como usar EEPROM do Arduino?
Inclua #include <EEPROM.h>. Salve com EEPROM.write(addr, val).
Como usar millis() em vez de delay()?
Evita travar o código. Ex: if (millis() - ultimaVez >= 1000) { ... }.
Como usar sensor de cor TCS3200?
Selecione filtro (S2/S3), leia frequência com pulseIn().
Como usar shield Ethernet?
Use biblioteca Ethernet. Configure IP fixo ou DHCP.
Como usar sensor de inclinação?
Saída digital: HIGH quando inclinado. Use como botão.
